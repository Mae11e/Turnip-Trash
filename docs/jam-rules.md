# Mini Jam 202 - Règles et Informations

## Format
- **Durée** : 72 heures
- **Fréquence** : Toutes les 2 semaines sur Itch.io

## Thème & Limitation

### Thème : Power-up
- Purement inspirationnel (pas obligatoire)
- Interprétation libre

### Limitation : Ridiculously Overpowered
- **OBLIGATOIRE**
- Interprétation libre
- Si tu crois en ton interprétation, d'autres y croiront aussi

## Critères de Notation (5 jours après la jam)

| Critère | Description |
|---------|-------------|
| **Enjoyment** | Plaisir de jeu |
| **Concept** | Originalité/intérêt de l'idée principale |
| **Presentation** | Qualité d'exécution |
| **Use of limitation** | Intégration de la limitation |

> Note : Ne pas juger les jeux sur l'utilisation du thème, seulement la limitation compte.

## Soumission
- Si retard : annonce sur Discord + post dans l'onglet communauté de la jam
- Possibilité de soumettre en retard
